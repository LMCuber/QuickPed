# How to run
```bash
zig build run
```
or
```bash
zigup run 0.13.0 build run
```
if you have [zigup](https://github.com/marler8997/zigup) installed.

This project uses the [zig-raylib-imgui-template](https://github.com/schmee/zig-raylib-imgui-template), and uses `zig`, `raylib`, `ImGui`, `raylib-zig`, `zgui`, `rlImGui`.
